# Before You Install
1. If you do not have Visual Studio 2022, open and follow [the official microsoft visual studio 2022 installation guide](https://learn.microsoft.com/en-us/visualstudio/install/install-visual-studio?view=vs-2022), then come back here.
2. Open Command Prompt and run the following command `dotnet --list-sdks`.
3. If there is a dotnet sdk installed that begins with 7.(something), you are ready to install Hopscotch On Desktop. If not, follow the instructions below.
4. Open [the official microsoft dotnet 7.0 webpage](https://dotnet.microsoft.com/en-us/download/dotnet/7.0) and install any version of dotnet 7.0.
5. Follow any instructions that the dotnet installer gives you.
5. Go back to step 2 to confirm that it was a successful installation.
